A lightspeed toolchain manager for languages and their dependency managers.

Supports Deno, Node.js (npm, pnpm, yarn) and Go.

Get started
  • # Install proto
  • curl -fsSL https://moonrepo.dev/proto/install.sh | bash
  • # Install Deno
  • proto install deno 1.31
  • # Use immediately
  • deno run index.ts

All in one

  • Lightspeed commands

    Download, install, and run tools with lightspeed, thanks to our Rust based foundation.

  • Universal toolchain

    Manage multiple languages and dependency managers through a single command.

  • Ecosystem aware

    Detects and infers from a language's ecosystem for maximum compatibility.

    Configure tools and their versions per directory, or per project.
