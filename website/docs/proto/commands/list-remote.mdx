---
title: list-remote
---

The `proto list-remote <tool>` (or `proto lsr`) command will list available versions by resolving
versions from the tool's remote release manifest.

```shell
$ proto list-remote node
...
18.10.0
18.11.0
18.12.0
18.12.1
18.13.0
18.14.0
18.14.1
18.14.2
19.0.0
19.0.1
19.1.0
19.2.0
19.3.0
19.4.0
19.5.0
19.6.0
19.6.1
19.7.0
```

### Arguments

- `<tool>` - Type of tool.
